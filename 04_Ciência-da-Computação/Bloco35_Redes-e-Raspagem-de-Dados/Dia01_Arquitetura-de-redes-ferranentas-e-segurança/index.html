Conheça nossa formação

Uma formação completa, intensiva e com aprendizado na prática para quem quer seguir a carreira de desenvolvimento web.

Módulo 1:Fundamentos Desenvolvimento web

Ao final do módulo você será capaz de desenvolver páginas estáticas e dinâmicas funcionais utilizando as linguagens HTML, CSS e JavaScript, aplicando todas as propriedades necessárias para criar layouts de páginas semânticas e responsivas, de forma a se adaptar a vários tamanhos de tela.

Você irá utilizar o JavaScript para desenvolver seus projetos, descobrindo o poder dessa linguagem e adquirindo sólidos conhecimentos nas partes fundamentais do código, exercitando o raciocínio lógico e aplicando seus conhecimentos em casos reais.

Você também irá dar os seus primeiros passos para implementação de testes unitários, conhecendo na teoria e na prática a importância desta etapa no desenvolvimento dos seus projetos. Você irá aprender também a utilizar o “Git” para desenvolver seus projetos, sendo essencial para o gerenciamento de versões de código, utilizando-o para durante o desenvolvimento dos projetos através de todo o curso.

Além disso, terá seu primeiro contato com o desenvolvimento de soft skills em momentos guiados pela nossa equipe, de forma a te preparar para o início da carreira como pessoa desenvolvedora.

Módulo 2:Desenvolvimento Front-end

Ao final do módulo, você será capaz de criar aplicações front-end sólidas, bem estruturadas, baseadas em dados estáticos ou consumindo informações de APIs externas, com ou sem gerenciamento interno de estado, utilizando ReactJS e várias tecnologias desse ecossistema.

Você também será capaz de criar rotinas bem estruturadas de teste para as suas aplicações, utilizando testes unitários assíncronos baseados no comportamento do usuário para garantir a renderização e o comportamento esperado.

Módulo 3:Desenvolvimento Back-end

Ao final do módulo, você será capaz de usar Docker pra colocar no ar aplicações Node.js com Express, com e sem Typescript, de ponta a ponta: do recebimento de uma requisição ao registro de informações num banco de dados, MySQL ou MongoDB, e o retorno para quem fez a requisição.

Você fará APIs RESTful e outras aplicações usando Programação Orientada a Objeto, Arquitetura em camadas e fazendo testes unitários e de integração para deixá-la tão bem organizada e cuidada quando possível!

Módulo 4:Introdução à Ciência da Computação

Ao final do módulo você será capaz de usar Python para construir algoritmos orientados a objetos e seguindo diversos padrões de projeto.

Além disso será capaz de construir ferramentas capazes de coletar dados de páginas na Web (Raspagem de Dados). Serão apresentados conceitos relacionados à análise de complexidade de algoritmos (BigO Notation), além das principais estruturas de dados.
